C. 对话 / 工作台 / 成果三模式

三者成为同级模式，默认进入对话；桌面与移动端结构一致，利用标签切换完整空间。

  • 响应式最简单
  • 每个区域都能使用完整宽度
  • 切换会割裂输入、状态与成果的联系
