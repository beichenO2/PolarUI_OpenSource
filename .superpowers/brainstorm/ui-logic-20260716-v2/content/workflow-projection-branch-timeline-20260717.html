Branch 基石：Stage 是 Checkpoint 投影，修改历史就是新时间线

Workflow 状态沿 Checkpoint 追加。Stage 只帮助人理解某个 Checkpoint；真正的分叉锚点永远是不可变的 Checkpoint，而不是 Stage 名称。
